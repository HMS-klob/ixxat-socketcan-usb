# Support

## How to file issues and get help

This project uses [GitHub issues][gh-issue] to track bugs and feature requests. Please search the existing issues before filing new issues to avoid duplicates. For new topics, file your bug or feature request as a new issue.
As an alternativ you can contact our customer support via the [HMS support portal][supportportal].

For help and questions about using this project, please look at the [readme][readme] and our [Contributor's Guide][contributor].
